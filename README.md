# Setup
```
git clone https://github.com/toddpla/proof-of-work.git
cd proof-of-work
yarn install
```

# Testing
### Unit
```
yarn test
```
### Integration
```
yarn run cypress:open
```

# Deployment
```
yarn build
firebase deploy
```
